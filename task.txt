file2 содержит поля responseToken, docType

Необходимо проанализировать несоответствия в результатах. 
По responseToken из file1 найти значение в file2. Сравнить docType из этих двух файлов.

второй уже детальное описание несоответствие в формате json.

таким образов в третьем файле должно быть 5 столбцов
со следующиеми хедерами - operationToken, responseToken, docType, isIncorrect, detailed

Создать четвертый файл с хедерами из третьего файла, записать все строки с isIncorrect = True
чтобы записать в формате json используй библиотеку json


Это уже с библиотекой requests
1. Есть страница с открытым (доступно всем) перечнем террористов. 
Ссылка: https://fedsfm.ru/documents/terrorists-catalog-portal-act
2. Необходимо получить html страницу с помощью HTTP GET запроса (используй например urllib, requests)
3. Выделить из полученной строки часть с Физическими лицами (весь список от начала до конца)
4. Записать в файл выделенное.

Усложненный вариант:
1. Повторить 1,2,3;
2. Разделить полученную информацию построчно в массив
3. Имея на руках массив с данными записать, удалив html-теги, строки в файл через точку с запятой ";": пример:
НОМЕР;ИМЯ;ФАМИЛИЯ;ОТЧЕСТВО;ДАТА РОЖДЕНИЯ;ОСТАЛЬНАЯ ИНФОРМАЦИЯ
Примечание: имея массив строк - выполнять надо в цикле, регулярные выражения если нужны - проси, если найдешь
способ обойтись без них - похвально.
